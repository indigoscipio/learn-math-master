1.5 Subtraction

Let a and b be any numbers.
Subtraction a - b is defined as

a - b = a + (-b)

Subtracting a number just means adding its opposite.

Problem 1.21
Use the definition of subtraction to answer the following questions:

b. What is 17 - 17?
c. What is 17 - 0?

Answer:
By the definition of subtraction,
the subtraction of a - b is defined as
a - b = a + (-b)

a. 0 - 17
   = 0 + (-17)
   = -17

b. 17 - 17
   = 17 + (-17)
   = 0

c. 17 - 0
   = 17 + (-0)
   = 17

Problem 1.22
Use the definition of subtraction to answer the following questions:
a. 11 - (-13) = 11 + 13
b. -11 - 13 = -(11 + 13)
c. 11 - 13 = -(13 - 11)

By the definition of subtraction,
the subtraction of a - b is defined as
a - b = a + (-b)

a. LHS -> 11 - (-13)
   = 11 + (-(-13))
   = 11 + 13

b. LHS -> -11 - 13
   = -11 + (-13)
   = -1(11 + 13) by distributive property
   = -(11 + 13)

c. RHS -> -(13 - 11)
   = -(13 + (-11))
   = -13 + -(-11)
   = -13 + 11
   = 11 - 13 by commutative property

================================

Important Properties:
subtraction of negation: x - (-y) = x + y
subtraction from negation: -x - y = -(x + y)
negation of subtraction: -(x - y) = -x + y = y - x

1.6 Reciprocals

Identity for summation = 0

Identity for multiplication = 1

1.8.11
Given numbers a and b, let a O (operation) b equal 2a + 2b.
For example, 3 O 4 equals 14.

a. Show that O is commutative.

To check commutativity, we need to show
a O b = b O a by definition of commutative property.

Since we know that a O b = 2a + 2b, we need to show b O a = 2b + 2a.

From the example we have:
3 O 4 = 3(2) + 4(2) = 6 + 8 = 14

Check for commutative:
4 O 3 = 4(2) + 3(2) = 8 + 6 = 14
Still equals.

The commutative property of addition states for any numbers x and y,
x + y = y + x.
Since 2a and 2b are numbers, we can conclude that 2a + 2b = 2b + 2a.
Therefore, a O b = b O a. The operation is commutative.

b. Show that O is not associative.

To show O is associative, we must show
(a O b) O c = a O (b O c)

LHS:
2a + 2b O c
2(2a + 2b) + 2c
= 4a + 4b + 2c

RHS:
a O (2b + 2c)
= 2a + 2(2b + 2c)
= 2a + 4b + 4c

Therefore the two sides are not equal, and not associative.

1.8.16
Given numbers a and b, define a O b to be a + ab + b.
For example, 2 O 3 = 2 + 2(3) + 3 = 11.

a. Show that the operation O is commutative.
b. Show that O is associative.
c. What is the identity of O? That is, what is the identity of the
   number I such that x O I = x for all values of x?
d. What is the inverse of 1 with respect to O, that is, what number goes in the
   blank to solve __ O 1 = I, where I is the number that you found in part c?

Answer:
a. To show commutativity, we must show
   a + b = b + a for all numbers a and b.

   Since we know a O b = a + ab + b and 2 O 3 = 2 + 2(3) + 3,

   we need to show b O a = b + ba + a.

   By commutative multiplication, ba = ab.
   By commutative addition, a + ab + b.
   Therefore two sides are equal.

b. To show a O b is associative, we need to prove
   (a O b) O c = a O (b O c)

   LHS
   = (a + ab + b) O c
   Let x = (a + ab + b)
   = x O c
   = x + xc + c
   Substitute back:
   = (a + ab + b) + (a + ab + b)c + c
   = (a + ab + b) + (ac + abc + bc) + c
   = a + b + c + ab + ac + bc + abc

   RHS
   a O (b O c)
   = a O (b + bc + c)
   Let y = (b + bc + c)
   = a O y
   = a + ay + y
   = a + a(b + bc + c) + (b + bc + c)
   Substitute back:
   = a + (ab + abc + ac) + (b + bc + c)
   = a + b + c + ab + ac + bc + abc

   Therefore the two sides are equal, associativity is proven.

c. Number I such that x O I = x

   x + xI + I = x
   Subtract both sides by x:
   xI + I = 0
   = I(x + 1) = 0 by multiplicative identity property

   I must be 0 for this to equal 0.
   Therefore the identity element is 0.

d. Finding the inverse:
   a O 1 = I
   Since we found the identity = 0, we can substitute:
   a O 1 = 0

   a + a1 + 1 = 0
   a + a + 1 = 0
   2a + 1 = 0
   a = -1/2
